# GED path 

GED is a lightweight, experimental, modular game engine built in OpenGL 3.3 designed for building 2D and 3D interactive applications with a focus on simplicity, performance, and flexibility.

---

## note !
This will be a temporary repository until the game engine is done.
This Engine is still in development!!! if there are errors then you can create a issue and then i will try to solve it.

---

## 🚀 Features

- Cross-platform architecture
- Modular rendering system
- Entity-component system (ECS)
- Real-time physics support
- Scene management
- Input handling (keyboard, mouse, gamepad)
- Asset pipeline for textures, models, and audio
- Scripting support (planned / optional depending on your implementation)

---
## 🚀 Requirements

- GPU that supports OpenGL 3.3 or higher
- Drivers that supports OpenGL 3.3 or higher
- Windows or linux (I have only tested Windows 11 & Ubuntu 26.04)
